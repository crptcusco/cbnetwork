# Coupled Boolean Network Library

This library contains resources for the creation and analysis of Coupled Boolean Networks (CBN). The code was
developed during the author's PhD in Computer Science at UFABC and is made available to help researchers reproduce
and extend the experiments from that work.

If you use this library in published research, please cite the original thesis or related publications.

Quick setup (Ubuntu example)
-----------------------------
Prerequisites and example setup commands for Ubuntu 22.04:

```bash
sudo apt update
sudo apt install -y minisat python3-distutils
python3 -m pip install --upgrade pip
pip install -r requirements.txt
```
